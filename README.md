## 🚀 The Vision
Most young people in Kenya shy away from farming due to the "Jembe Trauma" and lack of capital. **Rahisi Agro Pay** flips the script. We don't just provide loans; we provide a **Startup Toolkit** for the soil. 

By turning a smartphone into a "Digital Tractor," we allow youth to walk into any Agrovet, scan a Till number, and walk out with high-yield inputs on credit—instantly.

### ✨ Key Innovator Features
* **Agri-Trust Score:** A gamified reputation system that replaces scary traditional credit scores.
* **Live ROI Projector:** Shows farmers their projected harvest value (e.g., 250% ROI) before they even checkout.
* **Micro-Insurance Toggle:** One-tap climate & pest insurance built directly into the checkout flow.
* **Sliver-Design Dashboard:** A premium, banking-grade interface designed for the modern user.
* **High-Security OTP:** Frictionless login powered by Africa's Talking SMS API.

## 🛠 Tech Stack
* **Frontend:** Flutter (Mobile Framework)
* **Backend:** FastAPI (Python High-Performance API)
* **Cloud Hosting:** Render (USA-based infrastructure)
* **Messaging:** Africa's Talking API (Kenyan Telecom Gateway)
* **CI/CD:** GitHub Actions (Automated Builds & Testing)

## 📦 Getting Started for Testers
1.  **Download the APK:** Navigate to the [Actions Tab](https://github.com/YOUR_USERNAME/rahisipay_app/actions), click the latest successful build, and download the `rahisipay-debug-apk`.
2.  **Install:** Transfer the `.apk` to your Android device and allow "Install from Unknown Sources."
3.  **Login:** Enter your M-Pesa number to receive a secure 4-digit PIN.
